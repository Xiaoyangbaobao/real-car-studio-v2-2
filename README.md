# Real Car Studio v2

A modular Three.js + TypeScript + Parcel starter for a cinematic car studio.

## Run

```bash
npm install
npm run dev
```

## Replace with your own assets

- Put your car model at `static/models/car.glb`
- Put HDR files at:
  - `static/hdri/desert.hdr`
  - `static/hdri/beach.hdr`
  - `static/hdri/snow.hdr`

The project runs even without those assets:
- it falls back to a procedural concept car
- it falls back to color/sky style environments
